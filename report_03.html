03 — 新兴科技专项：GPU 算力卡经营租赁风险深度分析

一、业务模型

1.1 核心逻辑

新兴科技（天保旗下）
    │
    ├── 采购 NVIDIA B300 GPU 服务器
    │   资金来源: 集团支持 + 融资
    │
    ├── 经营租赁给 汇天（或其他算力运营商）
    │   租赁模式: 5年闭口合同
    │   租金: 参照市场价（如B300集群 19万/台/月）
    │
    └── 汇天再将算力转租/提供Token服务给终端客户
        终端客户: 北京移动/互联网大厂/网游公司
        Token服务: 5年闭口合同

  • 5090: 35亿/5万卡 ≈ 7万/卡

B300 静态回收期: 102/29 = 3.5年 (5年闭口合同, 剩余1.5年为纯利润期)

更深层的风险

GPU租赁市场的核心矛盾: 技术进步越快, 旧卡贬值越快。

如果大模型架构发生根本性变化（如从Transformer转向新架构），可能导致:

  • 旧卡无法高效运行新模型 → 算力需求结构性变化
  • 特定卡型被「淘汰」→ 残值接近零

但关键缓冲: 5年闭口合同

榆林合作模型中的「5年闭口合同」是关键保护:

  • 租金在5年内锁定, 不受Token价格波动影响
  • 客户(北京移动/互联网大厂)承担了Token价格风险
  • 新兴科技的风险仅在「合同到期后能否续约/找到新客户」

评估: 如果有真实客户背书的5年闭口合同, Token价格风险被有效隔离。但需要验证:

  • 合同是否真正的「闭口」(不可撤销/不可单方面降价)？
  • 客户信用质量如何？
  • 违约条款是否充分保护出租方？

分化趋势

市场正在分化:

  • 高端训练算力(B300/H200级别): 供应受出口管制限制, 可能维持紧缺
  • 中低端推理算力(5090/国产卡): 供应充足, 价格下行压力大
  • 特殊场景算力(金融/政务/军工): 安全性要求高, 合规IDC有溢价

🟡 风险5：大模型技术路线变化

潜在冲击

最差情景: 5年后B300可能因架构不兼容而无法运行主流模型 → 残值趋近于零。

闭口合同必须包含:

  1. 不可撤销条款（客户不能因Token价格下降而提前终止）
  2. 违约金条款（提前终止赔付剩余租金≥80%）
  3. 母公司担保（终端大客户的母公司提供履约担保）
  4. 设备所有权归属（租期内/期满后设备的归属和处理方式明确）

4.2 第二道防线：采购策略

策略 说明
分批采购, 不一次满仓 先小批量(如1000卡)试点, 验证终端需求后分批追加
多品牌混合 NVIDIA + 国产(华为昇腾) + 其他(AMD MI300X) 分散供应商风险
合规优先 采购前确认BIS出口许可证, 不使用灰色渠道
期货+现货结合 部分锁定期货价格, 部分随行就市, 降低价格风险
残值对赌 与OEM厂商约定最低回购价或以旧换新方案

4.3 第三道防线：运营策略

策略 说明

算力经纪 不全部做长租, 留20-30%做短租/按需, 捕捉高毛利机会
残值管理 建立GPU残值预测模型, 在最佳时点处置旧卡

4.4 第四道防线：金融对冲

工具 用途

5.2 做多大规模？

阶段 规模 投资 风险 条件
试点 500-1000卡 5-10亿 可控 B300可交付+终端合同签署
扩展 1000-3000卡 10-30亿 中等 试点成功+客户追加需求

推荐: 从试点开始（500-1000卡），12个月试点期后评估是否扩展。新兴科技 GPU 业务规模上限为几千卡级别（不超过 5000 卡），不以万卡级为目标。

Plan B选项 优势 劣势
NVIDIA B20（中国特供版） NVIDIA生态兼容 性能阉割, 竞争力下降

推荐: 同步考察华为昇腾+AMD方案, 保持至少2个供应商选项。

六、总结

新兴科技的GPU经营租赁是一个高风险、高回报的业务:

  • 高回报: B300 5年租金收入约145万/卡 vs 成本102万/卡, 利润率~30-40%（若合同顺利履行）
  • 高风险: 出口管制是最大的不确定性, 其次是硬件迭代和产能过剩

核心建议:

  1. 先解决合规问题: B300能否合规进入中国是「能不能做」的前提
  2. 必须背靠背: 没有闭口终端合同, 不采购
  3. 从小做起: 试点500-1000卡, 不要一次投入百亿级
  4. Plan B就绪: 国产替代方案同步准备
  5. 残值管理: 合同中加入残值保护条款
