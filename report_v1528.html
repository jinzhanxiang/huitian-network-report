1.4 纵轴三：治理端——34% 份额的控制力与回报

1.4.1 现状权力边界

持有人大会（最高权力机构）：

  • 天保 34% 投票权 < 1/3 → 无法单方否决重大事项
  • 重大事项通常需 2/3 以上通过 → 34% =「重大影响」而非「控制」

经管委（常设决策机构，5 票制）：

  • 管理人 2 票 + 汇天 1 票 + 投资人（天保）1 票 + 运管 1 票 = 5 票
  • 天保 1/5 票 = 20% → 不足以主导运营决策
  • 管理人+运管联合 = 3/5 = 60% → 过半

现状判断：34% 份额在当前治理架构下 →「重大影响」→ 权益法核算 → 不能并表

1.4.2 并表升级路径：A+C+D 联动

仅靠 34% 份额不足以并表，但通过治理条款组合可构建实质控制（《企业会计准则第 33 号》第七条三要素）：

路径 操作方式 并表强度 实操性
路径 A: 经管会席位主导 天保 34% = 最大份额持有人 1 票 + 一票否决条款补足 强 高
路径 C: 经管会一票否决 章程写入"重大事项需最大份额持有人书面同意" 强 高
路径 D: 一致行动协议 天保+汇天+运管方三方签"重大事项一致行动"备忘录 中强 高

A+C+D 组合的并表论证（不依赖汇天投票权委托）：

  • 要素一（权力）✅：34% 最大份额持有人 + 一票否决条款 + 运管方协同 → 实质单方主导
  • 要素二（可变回报）✅：34% 现金流（半年分红）+ 运管费收入 → 回报量与份额比例匹配
  • 要素三（权力影响回报）✅：经管会实质主导权 + 重大事项决策权 + 运管机构任命权

2.1 战略协同必要性
